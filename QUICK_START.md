## Setup in 3 Steps

### 1. Create Accounts
Go to `/signup` and create each account with the credentials above.

### 2. Verify Driver
- Login as **driver** → Upload 6 documents → Request verification
- Login as **admin** → Go to Drivers → Verify the driver

### 3. Test Booking
- Login as **passenger** → Book a ride
- Login as **driver** → Accept the ride → Complete it

## URLs

- **Passenger Dashboard**: http://localhost:3000/passengers

## Required Documents for Driver

1. Government ID
2. Selfie
3. Driving License
4. Car RC (Registration Certificate)
5. Number Plate Photo
6. Car Photos
